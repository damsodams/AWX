# AWX
Installation d'Ansible-AWX l'alternative gratuite de ansible-tower sur CentOs7
avec docker-compose.

## Pre-requis 
#Matériel:
  - 4GB RAM.
  - 20 GB d'espace dédié sur le disque dur.
  - Processeur 64 Bit
  - 2 CPU minimum.
#Logiciel:
  - PostgreSQL version 10 
  - Ansible version 2.2
